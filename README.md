## ✨ KEY HIGHLIGHTS

*   **🎮 INTERACTIVE GAMEPLAY:** Engaging "Heart Catch" mini-game built with low-latency touch controls.
*   **📱 MOBILE-FIRST ENGINE:** Utilizes **CSS Hardware Acceleration (GPU)** for lag-free animations on all devices.
*   **💎 PREMIUM UI/UX:** Modern **Glassmorphism** design with real-time backdrop-blur effects.
*   **⚡ ULTRA LIGHTWEIGHT:** 100% Vanilla JavaScript & CSS3 — **Zero external libraries or dependencies.**

## 🛠️ CUSTOMIZATION GUIDE (PERSONAL USE)

**Adapt this project for your loved ones in two simple steps:**

### 1️⃣ UPDATE THE VISUALS
Place your custom photo in the project directory and update this line in `index.html`:
```html
<!-- Locate this line and change 'sister.jpeg' to your image filename -->
<img src="your-photo.jpg" alt="Celebration Image" class="sister-img">

```

### 2️⃣ PERSONALIZE THE MESSAGE

Locate the `wishes-box` class in the HTML code and type your heart-felt congratulations directly into the text area.

## ⚙️ ADVANCED PERFORMANCE OPTIMIZATIONS

**To maintain maximum smoothness on mobile hardware, the following were implemented:**

* **`will-change: transform`**: Pre-renders animation layers directly on the GPU.
* **Touch-Action Optimization**: Removes 300ms input delay for instant interaction.
* **Element Throttling**: Automatically limits active particles to prevent CPU overheating.
